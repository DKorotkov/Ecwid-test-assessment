1. Сборка:
   Установить node(npm) и gulp.
   Открыть командную строку и установить необходимые модули используя npm i
   gulp - собирает проект и запускает сервер
   gulp build - только собирает

2. Структура:
   index.html - Пустая страница с блоком для вывода галереи
   normalize.css - Обеспечивает для HTML-элементов лучшую кроссбраузерность в стилях по умолчанию
   style.css - минимальные стили для страницы
   main.js - скрипт, в котором вызываем Галерею

   gallery - основная папка с "библиотекой" Галереи
   gallery.js
      html/template.html - разметка html Галереи
      img - картинки Галереи
      scss
         gallery.scss - основной файл стилей
         _variables.scss - переменные
         _functions.scss - функции для простоты работы
         blocks - осноыне блоки (которые могут повторяться)

3. Работа:
   При нажатии кнопки "Загрузить" вызывается диалоговое окно для выбора файла JSON, работает только в случае если в поле ввода пустое.
   Если ввести в поле ввода ссылку на картинку и нажать "Загрузить", то она загрузится.
   Можно перетащить (drag&drop) файл с изображением в область Галереи.
